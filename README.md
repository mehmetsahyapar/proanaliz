# YaparExpertiz | Gayrimenkul Değerleme

Mehmet Yapar Güvencesiyle — Şanlıurfa odaklı yapay zeka destekli mülk değerleme platformu.

## Özellikler

- **Anlık Değerleme**: Mahalle, sokak ve mülk özelliklerine göre piyasa değeri tahmini.
- **Manuel Geofencing**: Karaköprü, Haliliye ve Eyyübiye için 34 mahalle poligonu (point-in-polygon).
- **Sokak Bazlı Mikrolokasyon**: Prestijli caddeler için şerefiye + yatırım puanı.
- **Foto Analizi**: AI Vision ile mülk fotoğraflarından kondisyon kademesi.
- **PDF Rapor & WhatsApp Paylaşım**: Tek tıkla rapor üretimi ve native paylaşım.

## Teknoloji

React 18 + Vite, Tailwind CSS, Leaflet, Supabase (Edge Functions, Postgres + RLS).

## Geliştirme

```sh
bun install
bun run dev
```
